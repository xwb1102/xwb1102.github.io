数据库的使用步骤

xwb
2018-06-09

实现数据库的使用，1、需要安装node/mongoDB，安装走默认。
2、启动数据库：
找到C:\Program Files\MongoDB\Server\3.4\bin

在这个磁盘下.\mongod –dbpath=数据库存放位置 –port=27017

3、使用 npm install 安装依赖
4、npm run dev 运行服务

3和4两步都是在数据库存放的位置里面进行的
5、在http://localhost:88 访问
（
如果下载速度较慢，可以尝试改变
npm i nrm -g
nrm test
nrm use cnpm
）
