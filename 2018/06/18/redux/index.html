Redux:
   reduer + Flux
Redux对于js应用而言是一个可预测状态的容器

Redux的核心概念有三个：
store；
reducers;
actions

一、安装
   npm i redux -D

二、引入
   import { CresteStore } from ‘redux’;

const reducer = function (state,action){

   switch(action.type){

四、创建store
   const store = createStore(reducer);
