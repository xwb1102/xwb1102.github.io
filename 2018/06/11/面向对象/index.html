面向对象

xwb
2018-06-11

首先面向对象有三个方法：抽象、封装、继承

继承有两种：
   拷贝继承
   属性继承-》
   类似继承：把父类放到子类下
     方法继承-》
   遍历父级的原型下的方法并且赋值给子类的原型

   原型继承
一、拷贝继承的使用
   首先，需要将属性拷贝下来，就是在新的构造函数里面，执行一下，需  要拷贝，并且是改变了this指向的原构造函数，这个函数的指向指着新的函  数，这样就可以将属性给拷贝过来了。
   其次，方法的拷贝。方法的拷贝，有以下几种方法：
1、对原构造函数进行for in ；就是将原构造函数的prototype给in给新的构造函数的prototype。
这里需要注意，就是，for in的时候，需要判断一下，传给新的构造函数的方法是不是原构造函数自身的东西，只有是他自身的，才给他放进去。
2、Object.assingn(Drag2.prototype,Drag.prototype)
使用浅拷贝的方式进行拷贝一份，注意，只能拷贝表层
3、Drag2.prototype = {…Drag.prototype,Drag.prototype,constructor:Drag2}
使用…的方式，将原构造函数展开，当然，因为这个是对象，所以，对象会将自己的构造函数也传给了Drag2，所以，对象里面需要设置一下，constructor:Drag2，将指针改正
     这样就可以完成拷贝了，接下来就进行其他需要的操作即可；
二、原型拷贝
  首先属性拷贝和属性拷贝是一样执行的
不一样的在于，方法的拷贝
原型拷贝，方法拷贝有以下几种方法：
   1、声明一个第三者，让原构造函数的prototype赋给新声明第三者的第三者的prototype，然后给第三者声明一个实例化对象将这个实例化对象赋给新构造函数的prototype；这样就完成了方法的拷贝。
// function Paohui(){}
// // Paohui.prototype = Drag.prototype;
// let ph = new Paohui;
// Drag2.prototype = ph;

   2、使用object.create(); 如：
let ph = Object.create(Drag.prototype);
ph.constructor = Drag2;
Drag2.prototype = ph;
这样也可以完成方法的拷贝。
