react的新建

xwb
2018-06-09

对于react框架的新建，只需要在项目下打开node，执行以下命令就可以了。
npm install -g create-react-app

create-react-app my-app

cd my-app

npm start

或者

npx create-react-app my-app

cd my-app

npm start

上面两种方法，第二种是针对最新版本node使用的，第一种通用。
