最近接触到了，hexo，也就是搭建个人博客，这个博客的话我了解到的是在GitHub上进行搭建这样的，可能别的
还有平台可以操作。今天在这里的话，简单叙述一下自己搭建博客的小小心得。搭建个人博客的话，首先，是准备工作，1、在GitHub上面创建一个以自己名字命名开始，以git.io结尾的仓库名;

例如：xwb1102.github.io；2、计算机上需装有git和node；有了这些就可以开始操作了
打开git(是git不是node)，输入命令 npm i hexo-cli -g 第二、 新建hexo环境 hexo init 文件名cd 进入文件 npm install 第三、hexo server (运行)http://localhost:4000 可以在本地打开博客;

部署：
   安装上传git的工具
   npm i hexo-deployer-git -S
然后找到环境中_config.yml配置链接
   deploy:
   type: git
   repo: 仓库地址
   branch: [branch]
打包：hexo g
   hexo d
清缓存
   hexo clean

git 部署
   hexo deploy

(在上面更改内容的时候，需要使用markdown，直接将需要添加的内容，以markdown的形式，放到里面去就可以了注意：要想博客能够改变，需重新打包，清除再部署)

修改主题：
  首先找到想要的主题，在主题文件夹themes下git clone 地址 ，将主题克隆下来，然后上传到git上也就是在这个文件夹上操作git push origin master；
打开第一个config文件，修改里面的主题，也就是theme后面的名字改成你克隆下来的主题的名字，然后打开git，hexo g ，hexo s 一下，查看一下本地是否成功，如果能成功，再hexo clean， hexo deploy，如果不行的话，就再下载一遍上传工具也就是
npm i hexo-deployer-git -S ,再重复hexo clean，hexo deploy;
